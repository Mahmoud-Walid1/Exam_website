# مجلد الأيقونات

هذا المجلد يحتوي على أيقونات المواد الدراسية التي يمكنك اختيارها عند إضافة اختبار جديد.

## الأيقونات الموجودة:

1. **math.png** - رياضيات
2. **arabic.png** - لغة عربية / لغتي
3. **science.png** - علوم
4. **english.png** - لغة إنجليزية
5. **social_studies.png** - دراسات اجتماعية
6. **islamic_studies.png** - دراسات إسلامية
7. **default.png** - أيقونة افتراضية لأي مادة

## إضافة أيقونات جديدة:

1. ضع ملف PNG في هذا المجلد
2. استخدم اسم وصفي واضح (مثل: physics.png, chemistry.png)
3. **المواصفات الموصى بها:**
   - مقاس: 400x400 بكسل (مربع)
   - تنسيق: PNG
   - ألوان: استخدم الكحلي (#1e3a5f) والسماوي (#0d9488)
   - تصميم minimalist واضح

الأيقونة الجديدة ستظهر تلقائياً في صفحة الأدمن!
